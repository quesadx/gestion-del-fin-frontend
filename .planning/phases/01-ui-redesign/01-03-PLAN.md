    **TacticalBackground.tsx** — CSS-only dynamic background system. Uses three layers: ambient gradient waves (CSS pseudo-elements), cursor glow (CSS radial gradient driven by JS mousemove), scanner sweep (CSS animation). NO Canvas. NO particles (deferred per research).

    Structure:
    ```typescript
    import { useEffect } from 'react';

    export function TacticalBackground() {
      // Cursor tracking: throttled mousemove → CSS custom properties
      useEffect(() => {
        let rafId: number;
        let lastUpdate = 0;
        const CURSOR_THROTTLE_MS = 50; // ~20fps

        const handleMove = (e: MouseEvent) => {
          const now = performance.now();
          if (now - lastUpdate < CURSOR_THROTTLE_MS) return;
          lastUpdate = now;
          rafId = requestAnimationFrame(() => {
            document.documentElement.style.setProperty('--gdf-cursor-x', `${(e.clientX / window.innerWidth) * 100}%`);
            document.documentElement.style.setProperty('--gdf-cursor-y', `${(e.clientY / window.innerHeight) * 100}%`);
          });
        };
        window.addEventListener('mousemove', handleMove, { passive: true });
        return () => { window.removeEventListener('mousemove', handleMove); cancelAnimationFrame(rafId); };
      }, []);

      return (
        <div className="pointer-events-none fixed inset-0 z-[-10] overflow-hidden" aria-hidden="true">
          {/* Ambient wave layer 1: cyan gradient blob, top-left, blur 80px, 20s drift */}
          <div className="absolute -top-[20%] -left-[10%] w-[70vw] h-[70vw] rounded-full bg-[radial-gradient(circle,rgba(6,182,212,0.12),transparent_55%)] blur-[80px] animate-ambient-drift-1" />
          {/* Ambient wave layer 2: blue gradient blob, bottom-right, blur 90px, 25s drift */}
          <div className="absolute -bottom-[20%] -right-[10%] w-[60vw] h-[60vw] rounded-full bg-[radial-gradient(circle,rgba(59,130,246,0.10),transparent_55%)] blur-[90px] animate-ambient-drift-2" />
          {/* Cursor glow: follows CSS vars, hidden on touch devices */}
          <div className="hidden md:block fixed inset-0 bg-[radial-gradient(600px_circle_at_var(--gdf-cursor-x,50%)_var(--gdf-cursor-y,50%),rgba(59,130,246,0.06),transparent_60%)] transition-opacity duration-300" />
          {/* Scanner sweep line */}
          <div className="fixed top-0 left-0 right-0 h-[2px] bg-[linear-gradient(90deg,transparent,var(--gdf-accent-secondary),transparent)] opacity-[0.15] animate-scanner-sweep" style={{ animationDuration: '8s' }} />
          {/* Reduced motion: wrap in @media (prefers-reduced-motion: reduce) by checking for the media query */}
        </div>
      );
    }
    ```

    For reduced motion support, add a useEffect that checks `window.matchMedia('(prefers-reduced-motion: reduce)')` and conditionally sets `--gdf-reduced-motion: 1` on :root. Then in CSS, wrap animations in `@media (prefers-reduced-motion: no-preference)`.

    **globals.css additions** (add to `@layer components` if not already added in plan 01-01):
    - If `.gdf-cursor-glow` and `.gdf-scanner-sweep` were already added in plan 01-01 Task 3, verify they exist. If not, add them.

    **Light mode consideration:** The background component renders the same structure in both modes — gradient colors use rgba with fixed alpha values. In light mode, the gradients are sufficiently subtle (0.12/0.10 alpha on light bg). No theme-specific variant needed.

    **Performance:** All three layers are GPU-composited (CSS transforms + blur). Zero JavaScript in the animation loop — only cursor tracking runs JS at 20fps. Total JS main-thread budget: < 2ms per frame.

    ```typescript
    import { create } from 'zustand';
    export type EmotionalState = 'stable' | 'alert' | 'critical';

    interface EmotionalStore {
      state: EmotionalState;
      manualOverride: EmotionalState | null;
      setState: (state: EmotionalState) => void;
      setOverride: (override: EmotionalState | null) => void;
      effectiveState: () => EmotionalState;
    }

    export const useEmotionalStore = create<EmotionalStore>((set, get) => ({
      state: 'stable',
      manualOverride: null,
      setState: (state) => set({ state }),
      setOverride: (override) => set({ manualOverride: override }),
      effectiveState: () => get().manualOverride ?? get().state,
    }));
    ```
    NO persist middleware — emotional state is session-only (derived from current camp data).
