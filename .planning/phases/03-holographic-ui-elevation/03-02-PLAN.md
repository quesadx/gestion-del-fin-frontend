    **Step B — AppShell scroll listener + z-index depth layers:**
    In AppShell.tsx, add a scroll-driven parallax effect per D-02 LOCKED:

    1. Import `useEffect, useRef` from React (already imported `useCallback, useState`).
       Add `useEffect` to the import and add `useRef`:
       ```typescript
       import { useCallback, useState, useEffect, useRef } from 'react';
       ```

    2. Inside the AppShell function component, after the existing hooks (around line 28),
       add a scroll handler that writes `--gdf-scroll-offset` to :root:

       ```typescript
       const contentRef = useRef<HTMLElement>(null);

       useEffect(() => {
         const el = contentRef.current;
         if (!el) return;

         const handleScroll = () => {
           const offset = Math.min(el.scrollTop * 0.5, 20); // max 20px parallax
           document.documentElement.style.setProperty('--gdf-scroll-offset', `${offset}px`);
         };

         el.addEventListener('scroll', handleScroll, { passive: true });
         return () => el.removeEventListener('scroll', handleScroll);
       }, []);
       ```
